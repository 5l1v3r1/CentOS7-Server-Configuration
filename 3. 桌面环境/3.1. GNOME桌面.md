## 3.1. GNOME桌面

`yum -y groups install "GNOME Desktop"` # 安装GNOME桌面环境

`startx` # 安装后运行

GNOME桌面环境启动。第一次引导，初始设置运行如下。

选择系统语言：

选择键盘类型：

![](../Contents/.png)

是否需要设置在线帐号：

![](../Contents/.png)

配置完成后，单击“Start using CentOS Linux”：

![](../Contents/.png)

GNOME桌面环境启动后：

![](../Contents/.png)

CentOS7的GNOME桌面默认以经典模式启动，如果要使用**GNOME Shell**，按以下设置：

`echo "exec gnome-session" >> ~/.xinitrc`

`startx`

GNOME Shell启动：

![](../Contents/.png)
