## 附0.1. 设置主机名

演示如何设置系统主机名（Hostname）。

更改主机名（如果重新启动系统，会恢复）：

`hostname` # 显示当前主机名

```
localhost.localdomain
```

`hostname dlp.srv.world` # 更改主机名

`hostname`

```
dlp.srv.world  # 已更改
```

永久更改主机名：

`hostnamectl set-hostname dlp.srv.world`
