## 附0.4. 设置键盘映射

设置系统的键盘（Keymap）映射，在下例中替换为自己的语言。

`localectl` # 显示当前状态

```
   System Locale: LANG=zh_CN.UTF-8
       VC Keymap: us
      X11 Layout: us
```

`localectl list-keymaps`

```
ANSI-dvorak
amiga-de
amiga-us
applkey
atari-de
atari-se
atari-uk-falcon
...
...
cn
...
...
uk
unicode
us
us-acentos
wangbe
wangbe2
windowkeys
```

`localectl set-keymap cn` # 设置

`localectl`

```
   System Locale: LANG=zh_CN.UTF-8
       VC Keymap: cn
      X11 Layout: cn
```
