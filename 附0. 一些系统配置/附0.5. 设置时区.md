## 附0.5. 设置时区

设置系统的时区（Timezone），在下例中替换为自己的时区。

`timedatectl list-timezones` # 显示时区列表

```
Asia/Aden
Asia/Almaty
Asia/Amman
Asia/Anadyr
Asia/Aqtau
Asia/Aqtobe
Asia/Ashgabat
...
...
Asia/Shanghai
...
...
Pacific/Rarotonga
Pacific/Saipan
Pacific/Tahiti
Pacific/Tarawa
Pacific/Tongatapu
Pacific/Wake
Pacific/Wallis
```

`timedatectl set-timezone Asia/Shanghai` # 设置

`timedatectl`

```
      Local time: Wed 2014-07-09 18:31:16 JST
  Universal time: Wed 2014-07-09 09:31:16 UTC
        RTC time: Wed 2014-07-09 09:31:15
       Time zone: Asia/Shanghai (CST, +0800)
     NTP enabled: yes
NTP synchronized: yes
 RTC in local TZ: no
      DST active: n/a
```
